# Technical Lead Workspace

Copy this folder to:

```text
~/.openclaw/workspaces/tech-lead
```

Then create the agent state directory:

```bash
mkdir -p ~/.openclaw/agents/tech-lead
```

Recommended convenience link:

```bash
mkdir -p ~/.openclaw/workspaces/tech-lead/links
ln -s ~/coding-projects ~/.openclaw/workspaces/tech-lead/links/coding-projects
```
