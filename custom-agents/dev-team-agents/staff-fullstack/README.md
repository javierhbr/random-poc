# Staff Fullstack Developer Workspace

Copy this folder to:

```text
~/.openclaw/workspaces/staff-fullstack-developer
```

Then create the agent state directory:

```bash
mkdir -p ~/.openclaw/agents/staff-fullstack-developer
```

Recommended convenience link:

```bash
mkdir -p ~/.openclaw/workspaces/staff-fullstack-developer/links
ln -s ~/coding-projects ~/.openclaw/workspaces/staff-fullstack-developer/links/coding-projects
```
