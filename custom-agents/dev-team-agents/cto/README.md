# CTO Guaripolo Workspace

Copy this folder to:

```text
~/.openclaw/workspaces/cto
```

Then create the agent state directory:

```bash
mkdir -p ~/.openclaw/agents/cto
```

Recommended convenience link:

```bash
mkdir -p ~/.openclaw/workspaces/cto/links
ln -s ~/coding-projects ~/.openclaw/workspaces/cto/links/coding-projects
```
