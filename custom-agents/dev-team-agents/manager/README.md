# Dev Team Manager Workspace

Copy this folder to:

```text
~/.openclaw/workspaces/dev-team-manager
```

Then create the agent state directory:

```bash
mkdir -p ~/.openclaw/agents/dev-team-manager
```

Recommended convenience link:

```bash
mkdir -p ~/.openclaw/workspaces/dev-team-manager/links
ln -s ~/coding-projects ~/.openclaw/workspaces/dev-team-manager/links/coding-projects
```
