# DevOps Engineer Workspace

Copy this folder to:

```text
~/.openclaw/workspaces/devops-engineer
```

Then create the agent state directory:

```bash
mkdir -p ~/.openclaw/agents/devops-engineer
```

Recommended convenience link:

```bash
mkdir -p ~/.openclaw/workspaces/devops-engineer/links
ln -s ~/coding-projects ~/.openclaw/workspaces/devops-engineer/links/coding-projects
```
