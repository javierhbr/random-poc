# Mobile Flutter Developer Workspace

Copy this folder to:

```text
~/.openclaw/workspaces/mobile-flutter-developer
```

Then create the agent state directory:

```bash
mkdir -p ~/.openclaw/agents/mobile-flutter-developer
```

Recommended convenience link:

```bash
mkdir -p ~/.openclaw/workspaces/mobile-flutter-developer/links
ln -s ~/coding-projects ~/.openclaw/workspaces/mobile-flutter-developer/links/coding-projects
```
