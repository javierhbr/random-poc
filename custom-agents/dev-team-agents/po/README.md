# Product Owner Workspace

Copy this folder to:

```text
~/.openclaw/workspaces/product-owner
```

Then create the agent state directory:

```bash
mkdir -p ~/.openclaw/agents/product-owner
```

Recommended convenience link:

```bash
mkdir -p ~/.openclaw/workspaces/product-owner/links
ln -s ~/coding-projects ~/.openclaw/workspaces/product-owner/links/coding-projects
```
