# Observe OPAL Codex Skill Pack

This pack converts the Observe OPAL documentation into a practical Codex/agent skill using a 3-layer context model:

- Layer 1: `SKILL.md` router
- Layer 2: `references/opal-authoring-playbook.md` skill logic
- Layer 3: focused resource/reference files

## Included references

- OPAL syntax
- OPAL data types and operators
- OPAL examples
- OPAL Functions
- OPAL Verbs
- Shape your data using stages

## Suggested install location

Place `observe-opal-skill/` inside your agent skills directory, for example:

```text
.skills/observe-opal/
```

or

```text
~/.codex/skills/observe-opal/
```

depending on your Codex / agent setup.
