# Observe OPAL Codex Skill

This skill pack uses a 3-layer context model:
- Layer 1: `SKILL.md`
- Layer 2: `playbooks/opal-authoring-playbook.md`
- Layer 3: focused references in `references/`

The helpful hints material from Observe is included in:
- `references/opal-helpful-hints-reference.md`

Suggested install pattern:
- keep this folder under your skills directory
- let the router open the smallest possible reference file
- avoid dumping all OPAL docs into every prompt
