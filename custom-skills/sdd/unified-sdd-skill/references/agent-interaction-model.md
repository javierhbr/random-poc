# Agent Interaction Model

The unified skill treats agents as helpers to phase owners.

```text
[Phase owner]
     |
     +--> BMAD agent support
     +--> OpenSpec artifact support
     +--> Speckit quality support
     |
     v
[Draft artifacts and recommendations]
     |
     v
[Human review and approval]
```

Default phase ownership:

- Platform -> Architect
- Route -> Team Lead
- Specify -> Product
- Plan -> Architect
- Deliver -> Team Lead

Default support pattern:

- BMAD helps with routing, planning depth, role framing, implementation framing, and review support
- OpenSpec helps with change package artifacts, apply, and archive
- Speckit helps with constitution, clarify, checklist, plan quality, task quality, and phased execution
