# Configuration

The first local configuration should declare:

- platform repo path
- default ref mode (`pinned`)
- allowed latest-query mode
- paths for specs, contracts, ADRs, and JIRA mapping
- cache directory
- read-only enforcement flags

See:

- `../../unified-sdd-methodology/templates/platform-mcp-boilerplate/platform-mcp-config.yaml`
- `../../platform-truth-mcp-server/examples/demo-platform-mcp-config.yaml`
