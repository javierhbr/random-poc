# Templates

## Constitution template
- Engineering principles
- Testing standards
- Reliability / security expectations
- UX / accessibility rules
- Performance priorities
- Documentation expectations

## Spec template
- Problem / opportunity
- Users / actors
- In-scope behavior
- Out-of-scope behavior
- Constraints
- Acceptance criteria
- Risks / assumptions

## Plan template
- Architecture overview
- Chosen stack / tools
- Components and responsibilities
- Data flow
- Interfaces / APIs
- Testing strategy
- Migration / rollout notes

## Tasks template
Each task should include:
- objective
- affected area
- dependencies
- implementation note
- validation note
- completion signal
